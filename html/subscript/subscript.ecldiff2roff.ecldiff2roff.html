subscript.ecldiff2roff.ecldiff2roff.parse_diff_dates(filename)¶

Read a text file with one date pair pr. line, each date separated by space.

The file can have dates in YYYYMMDD or in YYYY-MM-DD format, and the format can be mixed in the same file.

Lines starting with “–” or “#” are ignored, so are empty lines.

Example

20000101 20010101 2003-01-01 2004-09-01

Parameters

filename (str) – Existing file

Returns

A list of date pairs as tuples. The tuples contain

two strings, each string representing the date in YYYYMMDD format.
