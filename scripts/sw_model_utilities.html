SW_MODEL_UTILITIES¶

This is a simple interactive script for converting ‘a’, ‘b’ between normal and inverse Leverett SwJ formulations. This is in particular useful for RMS, which uses the inverse formulation while, input from petrophysicist is usually on the normal form. In addition, interactive plotting of Sw vs height is provided (simplified Leverett).

usage: sw_model_utilities [-h]

Interactive menu:

  1. Convert a TO A and b to B from Sw = aJ^b to Sw=(J/A)^(1/B)

  2. Convert A to a and B to b from Sw = (J/A)^(1/B) to Sw=aJ^b
